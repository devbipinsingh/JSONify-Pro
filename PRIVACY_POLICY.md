# Privacy Policy — JSONify Pro

Effective Date: May 2026

JSONify Pro respects user privacy and is designed to operate entirely locally within the user's browser.

## Information Collection

JSONify Pro does not collect, store, transmit, or share any personal information or sensitive user data.

The extension does not collect:

* Personally identifiable information
* Authentication credentials
* Financial information
* Browsing history
* Location data
* Health information
* Personal communications

## Local Data Storage

JSONify Pro may store the following data locally on the user's device using Chrome local storage:

* JSON history entries
* Editor preferences
* Theme settings
* Workspace configuration
* Open tabs and session state

All stored data remains exclusively on the user's device and is never transmitted to external servers.

## Data Processing

All JSON formatting, validation, parsing, minification, and visualization operations are performed locally within the browser.

No JSON content entered into the extension is uploaded, transmitted, or shared externally.

## Clipboard Access

JSONify Pro may use clipboard permissions solely to:

* Copy formatted JSON to the clipboard
* Paste JSON content into the editor

Clipboard data is processed locally and is never transmitted externally.

## Third-Party Services

JSONify Pro does not use:

* Remote servers
* Analytics services
* Tracking technologies
* Advertising networks
* External databases

## Remote Code

JSONify Pro does not execute remote code. All application code and assets are bundled directly within the extension package.

## Security

Because all processing occurs locally within the browser, user data remains under the user's control.

## Changes to This Policy

This Privacy Policy may be updated periodically to reflect extension improvements or policy changes.
